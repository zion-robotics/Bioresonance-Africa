## 🚀 Getting Started

### Prerequisites
- Node.js 16+ or Bun
- npm, yarn, or Bun package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/zion-robotics/Bioresonance-Africa.git
cd Bioresonance-Africa

# Install dependencies
npm install
# or
bun install
```

### Development Server

```bash
# Start development server (runs on http://localhost:8080)
npm run dev
# or
bun dev
```

### Build for Production

```bash
# Build optimized production bundle
npm run build
# or
bun run build

# Preview production build locally
npm run preview
# or
bun run preview
```

### Linting

```bash
# Run ESLint
npm run lint

# Build in development mode (with additional checks)
npm run build:dev
```

### Testing

```bash
# Run tests once
npm run test

# Run tests in watch mode
npm run test:watch
```


### Vite Config
The project uses Vite with React and SWC for fast builds. Key settings:
- Dev server: `http://localhost:8080`
- HMR overlay disabled for better UX
- Path alias: `@/` → `src/`

### Tailwind CSS
Customized Tailwind config with:
- Custom color scheme (navy, gold, foreground)
- Custom typography utilities
- Built-in CSS animations
